### 截图/复制即译，自动粘贴

[下载](https://github.com/HanyuuLu/CopyTranslatePaste/releases)

CopyTranslatePaste 是一个可以自动读取剪切板中的文字和图片进行翻译的一个工具；受困于经常要翻译各种格式和排版的论文，作者想要一个能够高效快速的翻译工具，求人不如求己，为了能让有类似需求的同学可以不再受到类似的困扰，作者开源了这个项目。
